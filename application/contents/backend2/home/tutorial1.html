productはすべての商品の一覧を表示します。

２.セッティング

タブのdashboard -> タブメニューのsettings -> サイドメニューのfrontendから設定できます。

frontend theme
テーマの入っているフォルダの名前を指定します。
ディレクトリのapplication->content->frontendの中に入っているのがテーマフォルダです。
デフォルトではbaseがはいっています。
テーマを変更するときはテーマフォルダ名をここにいれます。
lang
frontendの言語を指定します。
ja、en、en_USなど。
home page
最初に表示するページを指定します。
ウェブサイトの1つのページをderaemon-cmsではitem（アイテム）と呼びます。
itemは管理用の名前のsegmentを持っているのでこのsegmentで指定します。
site details
サイト全体で使う文字を設定していきます。
いくつも設定できます。
コロン「：」区切りでキーとバリューを作ります。
改行して追加します。

今回は以下のとおりに指定します。

  • frontend theme -> base（既定）
  • lang -> ja
  • home page -> home
  • site details ->
    name : うさぎ紳士服
    phone : 00 000 0000
    address : osaka japan

langは{{settings.lang}}で表示されます。

site detailsは{{settings.site.name}}のように「settings.site」のあとに「.」で設定した値をつないで表示します。

３.インデックスページ（ホーム）作成

最初に表示されるインデックスページ（ホーム）から作っていきます。

３-１.wrapperテンプレートの作成

templateタブ -> wrappersタブメニューを開きます。

フォームに

  • segment -> html
  • name -> HTML
  • content type -> text/html

と入力してcreateボタンを押す。

HTMLテンプレートがテーブルにできているのでeditボタンを押します。

contentに{{>division_content}}とだけ書かれています。
この中に次のテンプレートdivisionsがはいってきます。
ここに一番外側のhtmlを書いていきます。
そしてupdateボタンを押して保存します。

ブラウザー上で編集するのはめんどくさいので、直接ファイルを書き換えます。
テンプレートファイルはすべてテーマファイルの中にあります。
場所は、application->content->frontendのなかの先ほど設定したbasicフォルダの中にあります。
basicフォルダを開くと中にwrapperというフォルダがあり、その中にさっき作ったhome.htmlが作られています。 これをエディターで編集して保存するとブラウザ上にも反映されているので、リフレッシュしてください。

ファイルを次のように書いていきます。

<!DOCTYPE html>
<html lang="ja">
 <head>
  <meta charset="utf-8" />
  <title>ホーム - うさぎ紳士服</title>
  <meta name="keywords" content="紳士服,ジャケット,パンツ" />
  <meta name="description" content="ウサギの紳士服店です。" />
 </head>

 <body>
  <!--#wrapper-->
   <div id="wrapper">
    <!--division content-->
    {{>division_content}}
    <!--/division content-->
   </div>
  <!--#wrapper-->
 </body>
</html>

ここで、dタブのdashboard -> タブメニューのsettings -> サイドメニューのfrontendで設定した値を使っていきます。

{{>と}}に囲まれた部分が値になります。

  • <html lang="ja">の「ja」を{{settings.lang}}に変更します。
  • <title>ホーム - うさぎ紳士服</title>の「うさぎ紳士服」を {{site.name}}に変更します。

これでwrapperテンプレート完成です！

３-２.divisionテンプレートの作成

ページのdivisionを作成します。

templateタブ -> divisionタブメニューを開きます。

フォームに

  • wapper -> HTML
  • segment -> page
  • name -> PAGE

と入力してcreateボタンを押す。

ページテンプレートがテーブルにできているのでeditボタンを押します。

contentに{{>shape_content}}とだけ書かれています。
この中に次のテンプレートshape又はitemのcontentがはいってきます。
ここでは各dhibisionごとのhtmlを書いていきますが、特に使わないのでこのままにしておきます。
編集する場合は編集後updateボタンを押して保存します。

これも直接ファイルを書き換えることができます。
場所はさっきと同じく、application->content->frontendのなかのbasicフォルダの中、
divisionというフォルダがあり、その中にpage.htmlが作られています。

これでdivisionテンプレート完成です！

３-３.itemの作成

各ページごとに作っていくitemを作成します。

itemタブをクリックするとさっき作ったdivisionのpageができているのでクリックします。

フォームに

  • segment -> home
  • title -> ホーム
  • issued -> 2014-5-30

と入力してcreateボタンを押す。

issuedはこのitemを公開する日付を設定します。

ページテンプレートがテーブルにできているのでeditボタンを押します。

このpageインデックスで「order」、「is active」、「send comment is on」がまとめて設定できます。

editでも変更できます。

サイドバーに

  • check item home
  • index
  • edit
  • content
  • images
  • fields
  • received comments

のメニューがあるのでcontentをクリックします。

contentの中にcontentとだけ入っていいます。
これはダミーテキストなのでこれを削除して、htmlを書いていきます。
